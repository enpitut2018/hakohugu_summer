## ハコフグ feat. ゴンズイ玉 「CoMony」
## エレベーターピッチ
一人でタスクの進捗管理ができない筑波大生向けの相互の進捗チェックを可能にするアプリケーションCoMonyです。これは、位置情報を用いた、直接対面することによる強力な進捗の相互管理ができ、ToDoチェックアプリと違って複数人による互いへの強制力、およびモチベーションの大幅な上昇が見込まれます。
